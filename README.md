# 自动生成视频帧系统

## 使用步骤：
* （手动）先跟AI确认视频标题
* 安装 jd_main_image_video_crawler 浏览器插件
* 打开要带货的京东商品链接，使用浏览器插件抓取商品信息到固定的文件夹，比如D:\路由器\TP-LINK 7DR7280
* 使用 streamlit run app.py 启动系统
* 在左侧选择模板、录入标题
* 做右侧（路由器）文件夹里面选择之前下载好的商品文件夹
* 点击开始生产，等待生成即可
* 生成好的素材在output文件夹里面
* （手段）之后让AI生成封面图
* （手动）把封面图和素材图片放到剪映轨道上，口播搞在剪映里面用AI配音，之后把图片拉取到和AI配音相吻合的位置

>备注：这是目前使用的2.0版本，jd_spider.py文件目前没用上，这是1，0版本时候用到的，1.0版本是直接爬取京东参数列表的内容，但很不稳定，所以暂时取消了。
